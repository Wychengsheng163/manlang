服务案例

CLASSIC CASE

  • 1
    商业秘密类 TRADE SECRETS

    指不为公众所知悉、能为权利人带来经济利益，具有实用性并经权利人采取保密措施的技术信息和经营信息。
    2
    专利类 PATENT CLASS

    专利类司法鉴定通常包括三种，一种为判定涉嫌侵权产品或者方法是否属于发明或者实用新型专利的保护范围，第二种为判定涉嫌侵权产品与专利外观设计相同或者近似，以及专利无效中涉及的司法鉴定。
  • 3
    商标类 TRADEMARK CLASS

    商标类的司法鉴定通常为鉴定是否与权利人的注册商标相同或相近似。
    4
    版权类 COPYRIGHT CLASS

    即著作权，是指文学、艺术、科学作品的作者对其作品享有的权利（包括财产权、人身权）。
  • 5
    技术合同类 TECHNICAL CON

    技术合同类的司法鉴定通常包括三种，即技术开发合同纠纷的司法鉴定，技术转让成果的司法鉴定，和技术服务合同的司法鉴定
    6
    其他类 OTHER

    知识产权司法鉴定的其他类型包括是否属于不正当竞争等。
